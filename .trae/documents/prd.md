## 1. 产品概述

机车改装配件选配全站系统——为机车爱好者打造的在线改装零部件可视化选配平台，用户可按分类浏览配件、在机车模型上实时预览搭配效果、管理个人选配清单并查看总价。
- 解决机车改装配件选择困难、搭配效果不可视的问题，面向个人车主和改装店技师
- 降低改装试错成本，提升选配效率与决策信心

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 浏览配件、选配预览、管理清单 |
| 管理员 | 无需注册，本地使用 | 新增/编辑配件数据、管理分类 |

### 2.2 功能模块

1. **配件分类页**: 配件分类导航、配件卡片列表、筛选与搜索
2. **机车预览页**: 机车轮廓可视化、已选配件叠加预览、配件切换对比
3. **选配清单页**: 已选配件汇总、价格合计、清单导出、增删改操作

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 配件分类页 | 分类导航栏 | 按排气、轮毂、把手、灯组、车身套件等分类展示 |
| 配件分类页 | 配件卡片列表 | 展示配件图片、名称、价格、兼容性标签，支持筛选和搜索 |
| 配件分类页 | 配件详情弹窗 | 展示配件大图、规格参数、安装说明，支持一键添加到清单 |
| 机车预览页 | 机车轮廓展示 | 以侧视图SVG展示机车主体轮廓 |
| 机车预览页 | 配件叠加预览 | 已选配件在对应位置高亮/叠加显示 |
| 机车预览页 | 配件切换对比 | 点击不同位置切换当前配件，实时更新预览 |
| 选配清单页 | 清单列表 | 展示所有已选配件的名称、分类、单价、数量 |
| 选配清单页 | 价格汇总 | 实时计算配件总价，展示分类小计 |
| 选配清单页 | 清单操作 | 增删配件、修改数量、一键清空、导出清单 |

## 3. 核心流程

用户打开系统 → 浏览配件分类 → 选择感兴趣的配件 → 查看配件详情并添加到清单 → 在机车预览页查看搭配效果 → 调整搭配方案 → 在选配清单页确认最终方案 → 导出清单

```mermaid
flowchart TD
    "浏览配件分类" --> "选择配件"
    "选择配件" --> "查看详情"
    "查看详情" --> "添加到清单"
    "添加到清单" --> "机车预览"
    "机车预览" --> "搭配满意?"
    "搭配满意?" -->|"否"| "调整配件"
    "调整配件" --> "浏览配件分类"
    "搭配满意?" -->|"是"| "确认清单"
    "确认清单" --> "导出清单"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深碳黑(#1a1a2e) + 机车橙(#ff6b35) + 金属灰(#8d99ae)
- 辅助色：暗夜蓝(#16213e)、铬银(#e2e2e2)
- 按钮风格：圆角矩形、橙色主按钮带微光效果、暗色次按钮
- 字体：标题使用 "Orbitron" 科技感字体，正文使用 "Noto Sans SC" 中文衬线
- 布局风格：左侧固定分类导航 + 中间内容区 + 右侧悬浮清单面板
- 图标风格：线性描边图标，工业机械风

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 配件分类页 | 分类导航栏 | 深色竖向侧栏、橙色激活态、图标+文字标签 |
| 配件分类页 | 配件卡片列表 | 网格布局、深色卡片、悬浮放大效果、价格标签 |
| 配件分类页 | 配件详情弹窗 | 暗色遮罩弹窗、大图左侧、参数右侧、底部操作栏 |
| 机车预览页 | 机车轮廓展示 | 居中大SVG、深色背景、部件分区高亮 |
| 机车预览页 | 配件切换面板 | 底部横滑卡片条、选中态橙色边框 |
| 选配清单页 | 清单列表 | 表格布局、行悬浮高亮、删除按钮 |
| 选配清单页 | 价格汇总 | 固定底栏、橙色总价数字、导出按钮 |

### 4.3 响应式设计

- 桌面优先设计，1920px为基准宽度
- 平板适配：分类导航折叠为图标模式、卡片列表改为2列
- 移动端：分类导航变为底部Tab、卡片单列、清单全屏展示
